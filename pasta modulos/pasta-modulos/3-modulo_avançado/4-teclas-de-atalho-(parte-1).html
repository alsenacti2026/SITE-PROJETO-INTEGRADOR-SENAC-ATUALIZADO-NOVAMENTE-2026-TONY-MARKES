Teclas de Atalho (Parte 1)

Copiar, colar e desfazer atalhos evitam o esforço de usar o mouse para funções repetitivas:

Ctrl + C: Copia o texto ou item selecionado para a memória do computador.

Ctrl + V: Cola no local selecionado o texto ou item que foi copiado anteriormente.

Ctrl + X: Recorta o item (ou seja, copia apagando do lugar original para mover para outro espaço).

Ctrl + Z: Desfaz a última ação realizada (essencial para corrigir erros rápidos).
